# TFG: Análisis de supervivencia y factores pronósticos en pacientes con adenocarcinoma ductal pancreático: estudio retrospectivo de cohorte hospitalaria

Análisis reproducible en R aplicado a una cohorte hospitalaria de pacientes con adenocarcinoma ductal pancreático.

Contenido:

- Limpieza y depuración de datos clínicos
- Análisis descriptivo de las variables
- Definición del evento y censura para el análisis de supervivencia
- Construcción de la cohorte para análisis de supervivencia
- Curvas de supervivencia Kaplan-Meier. Y comparación entre grupos mediante test log-rank
- Análisis de supervivencia mediante modelos de Cox univariantes y multivariantes.
- Diagnóstico del supuesto de riesgos proporcionales

Generación de tablas y figuras listas para publicación

Software:

R

Nota sobre los datos:

Los datos clínicos originales no se incluyen en este repositorio por motivos de confidencialidad y protección de datos.
